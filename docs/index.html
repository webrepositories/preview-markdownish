How to Use the Markdown

Type markdown in the top editor. The rendered HTML preview appears below automatically.

Supported features:

Headings:
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

Bold:
**bold text**

Italic:
*italic text*

Strikethrough:
~~crossed out~~

Links:
[Link text](https://example.com)
 
Images:
![Alt text](https://example.com/image.jpg)

## Empty Lines
 

Bullet lists:
- Item
- Item
- Item

Numbered lists:
1. First
2. Second
3. Third

Tables:
| Name | Age |
|------|-----|
| Alex | 22  |
| Sam  | 30  |

Unlike standard Markdown, support has been added for " " to signify an empty line. This is necessary, because this tool does not always preserve spacing around elements as you might expect compared to other Markdown converters.

When printing, only the rendered HTML preview is shown, and it expands to fill the printable area.
